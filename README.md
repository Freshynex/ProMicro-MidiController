# Pro Micro Midi controller
A midi controller with buttons, potentiometers and display.

## Components

### Base
Standard breadboard 56x165mm 200/630pin

### MCU
AR-PROMICRO-TYPE-C (ATmega32U4)
[promicro]: https://cdn.sparkfun.com/assets/9/c/3/c/4/523a1765757b7f5c6e8b4567.png "Pro micro pinout"
![alt text][promicro]


### Keyboard
24 button keyboard by [bryckergithublink]

### Extra controls
2 Rotary pots
2 Sliding pots
2 buttons for octave switching
